# Palette

Canonical Budget Lab color palette. Single source of truth for all chart, table, and UI color decisions.

## Files

| File | Purpose |
|---|---|
| `colors.json` | Token source — names, hex values, roles. Read this first when writing code. |
| `colors.css` | CSS custom properties for web use (`var(--tbl-cat1)`, `var(--tbl-blue-400)`, etc.). |
| `colors-preview.png` | Rendered swatch grid for quick reference. |
| `_gen_preview.py` | Generator for the preview PNG. |

## Naming aliases

Users often refer to colors by common alternate names. Map these to the canonical name before resolving palette tokens:

| User says | Canonical |
|---|---|
| `purple` | `violet` |
| `pink` | `rose` |
| `yellow` | `amber` |
| `brown` | `russet` |

Aliases are also recorded in `colors.json` under `_naming_aliases` and per-color `aliases` arrays.

## Structure

### Brand
- `navy` (`#101F5B`) — primary brand color, headings, UI accents.
- `blue` (`#0072B2`) — links, default single-series data, interactive highlights. Same as `cat1_blue`.
- `sky` (`#63AAFF`) — light blue brand accent. Used in logo and brand UI.

### Categorical (7 hues, apply in order)
1. **Blue** `#0072B2`
2. **Amber** `#E69F00`  *(alias: yellow)*
3. **Violet** `#8856BF`  *(alias: purple)*
4. **Green** `#2A8B3A`
5. **Red** `#B8302C`
6. **Rose** `#CC79A7`  *(alias: pink)*
7. **Russet** `#7A5230`  *(alias: brown)*

Hue + luminance gaps are tuned for CVD safety (protanopia / deuteranopia), verified against Okabe-Ito (2008). When more than 4–5 series are needed, prefer small-multiples or highlight/dim over reaching deep into the palette.

### Tonal scales (8 tiers per hue)
Each categorical hue has an 8-tier monochromatic scale, with tiers named Tailwind-style (`50` lightest → `700` darkest). Tiers sit at fixed CIELAB L* values:

| Tier | L* | Typical use |
|---|---|---|
| 50 | 85 | Background fills, very-soft tints |
| 100 | 75 | Light fills, hover states |
| 200 | 65 | Soft fills, secondary backgrounds |
| 300 | 55 | Medium fills, gridlines on dark bg |
| 400 | 45 | Default stroke / mark / line — closest to most base hues |
| 500 | 35 | Emphasis stroke, dark text on light bg |
| 600 | 25 | Strong text, high-emphasis marks |
| 700 | 15 | Near-black text, deepest emphasis |

Scales were generated by holding hue + chroma at the categorical base and varying L* through the locked tiers. Chroma stays at the base value across all tiers (with tiny falloff at the L*=15 darkest tier where some hues exceed sRGB gamut).

### Structural (non-data)
Backgrounds, borders, gridlines, text colors. Defined in `colors.json` under `structural`.

Notable values for chart code:
- `axis_stroke` `#999999` (1 px) — the y=0 baseline gridline. Used wherever 0 (or the rebased index baseline like 100) appears in the y-range.
- `gridline` `#F0F0F0` (0.75 px) — standard horizontal gridlines.
- `annotation_dim` `#BBBBBB` — subtle reference lines, suppressed labels.
- `mark_black` `#000000` — neutral/baseline chart marks (national-average line, total line, reference threshold). Strong visual weight; use sparingly.
