# Logo Usage

## Color rule

The logo always uses one of two color pairings:

- **On light backgrounds**: `brand.navy` (`#101F5B`) for the wordmark + `brand.sky` (`#63AAFF`) for the accent stripe.
- **On dark backgrounds (knockout)**: `mark_knockout` (`#FFFFFF`) for the wordmark + `brand.sky` (`#63AAFF`) for the accent stripe.

The accent stripe is always `brand.sky`. Do not substitute other blues.
